Interactive URL Shortener Flowchart
Click on each step to expand and see the details about the URL shortening process.

User Inputs URL

The user enters a URL to be shortened. The URL is then validated to ensure it is in the correct format (e.g., http://).

Generate Short URL

A unique short URL is generated using a random string of characters. This is stored in the database along with the original URL.

Redirect to Original URL

When the user visits the short URL, they are redirected to the original URL. The visit is logged for analytics purposes.
